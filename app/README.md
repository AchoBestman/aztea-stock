# AzteaStock Desktop Application

Application SaaS multi-tenant de gestion Pharmacie & Supermarché. Conçu pour offrir une solution complète, sécurisée et évolutive pour la gestion quotidienne des pharmacies et supermarchés, AzteaStock permet aux entrepreneurs de piloter leurs activités avec précision, de garantir la conformité réglementaire et de servir leurs clients de manière optimale, que ce soit sur mobile, web ou desktop.
